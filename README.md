# Adventure-Works-Sales-Performance-Dashboard-Power-BI-
This project is a Power BI dashboard built using the Adventure Works dataset to analyze sales performance against budget.

<img width="587" height="332" alt="PowerBI Dashboard" src="https://github.com/user-attachments/assets/7df6689b-7cf5-4687-bd91-4074618760be" />

### Tools & Techniques Used
- Data Modeling: Designed a star schema with fact and dimension tables to optimize performance and enable scalable reporting.
- Power Query: Performed data cleaning, transformation, and shaping (handling nulls, formatting, merging tables).
- DAX: Created calculated measures for:
- Total Sales
- Budget
- Variance ($ and %)
- Power BI: Built interactive visualizations and KPI cards for executive-level insights.
### Key Insights
- Total Sales: $16.47M
- Budget: $16.87M
- Variance: -$395K (-2.3%) → Slight underperformance vs. target.
- Bikes category drives the majority of revenue but also contributes most to the negative variance.
- Sales trend increases toward year-end, with strong performance in Q4.
- Top-performing products are mainly from the Mountain-200 series.
### Business Value
This dashboard enables stakeholders to:
- Monitor budget performance in real time.
- Identify underperforming product categories.
- Track monthly sales trends.
- Analyze top-performing products and customers.
- Support data-driven decision making.
